# Groq SEO — Руководство по использованию

Полное руководство по всем AI-инструментам для SEO продвижения сайта **Freonn.ru** на базе Groq API.

---

## Быстрый старт

### 1. Добавить токен в Railway

Перейдите в Railway → ваш проект → Variables и добавьте:

```
GROQ_API_KEY=gsk_ваш_токен_здесь
```

**Важно:** Без этой переменной сайт работает в штатном режиме — никаких ошибок. AI-функции просто не активируются.

### 2. Проверить работу

После деплоя откройте в браузере:
```
https://freonn.ru/api/groq/status
```
Ответ `{"available":true}` — всё работает.

## Что добавлено на сайт

### AI-консультант (автоматически на всех страницах)

Кнопка с иконкой чата появляется в правом нижнем углу на всех страницах. Посетитель задаёт вопрос — получает ответ от AI, обученного на специфике Freonn.

**Что умеет:**
- Отвечать на вопросы о вентиляции, кондиционировании, отоплении, дымоудалении
- Называть телефон и предлагать оставить заявку
- Работать в режиме стриминга (ответ появляется постепенно)

**При недоступном Groq:** показывает статический ответ с телефоном — посетитель не замечает разницы.

## Скрипты для SEO (запускаются вручную)

Все скрипты находятся в папке `scripts/`. Запускаются из корня проекта.

### Скрипт 1: Генерация текстов для городских страниц

Генерирует уникальные LSI-тексты для 40+ городских страниц МО.

```bash
# Только сгенерировать и сохранить в JSON:
GROQ_API_KEY=gsk_xxx npx tsx scripts/generate-city-seo.ts

# Сгенерировать И автоматически обновить CityPage.tsx:
GROQ_API_KEY=gsk_xxx AUTO_PATCH=true npx tsx scripts/generate-city-seo.ts
```

**Результат:** `scripts/output/city-seo-texts.json`

**Что даёт:** 40+ уникальных страниц под запросы "монтаж вентиляции в [город]" вместо одного шаблонного текста.

---

### Скрипт 2: Генерация статей для блога

Создаёт полноценные SEO-статьи (700–900 слов) с мета-тегами.

```bash
# Сгенерировать все 10 статей из встроенного списка:
GROQ_API_KEY=gsk_xxx npx tsx scripts/generate-blog-articles.ts

# Сгенерировать одну статью на свою тему:
GROQ_API_KEY=gsk_xxx TOPIC="Монтаж чиллеров" CATEGORY="Кондиционирование" \
  npx tsx scripts/generate-blog-articles.ts --single

# Сгенерировать и автоматически добавить в BlogArticle.tsx:
GROQ_API_KEY=gsk_xxx AUTO_PATCH=true npx tsx scripts/generate-blog-articles.ts
```

**Результат:** `scripts/output/articles/*.md` + `scripts/output/articles-summary.json`

**Что даёт:** Регулярный свежий контент для блога без ручного написания.

---

### Скрипт 3: Генерация мета-тегов для всех страниц

Создаёт уникальные Title и Description для каждой страницы сайта.

```bash
GROQ_API_KEY=gsk_xxx npx tsx scripts/generate-meta-tags.ts
```

**Результат:** `scripts/output/meta-tags.json`

**Что даёт:** Уникальные мета-теги вместо шаблонных, улучшает CTR в поиске.

---

### Скрипт 4: Кластеризация семантики

Распределяет ключевые слова по страницам сайта, определяет интент.

```bash
# 1. Создать файл с ключевыми словами (по одному на строку):
mkdir -p scripts/input
nano scripts/input/keywords.txt

# 2. Запустить кластеризацию:
GROQ_API_KEY=gsk_xxx npx tsx scripts/cluster-keywords.ts
```

**Результат:**
- `scripts/output/keyword-clusters.json` — данные
- `scripts/output/keyword-clusters.md` — читаемый отчёт

**Что даёт:** Понимание, какие страницы нужно усилить, какие создать, какие ключи упущены.

## API эндпоинты (для разработчиков)

Все эндпоинты безопасны — при недоступном Groq возвращают fallback, не 500.

| Метод | URL | Описание |
|---|---|---|
| GET | `/api/groq/status` | Проверка доступности Groq |
| POST | `/api/ai/chat` | AI-чат (streaming + non-streaming) |
| POST | `/api/seo/meta` | Генерация мета-тегов для страницы |
| POST | `/api/seo/city-text` | SEO-текст для городской страницы |
| POST | `/api/seo/generate-article` | Генерация статьи блога |
| POST | `/api/seo/cluster-keywords` | Кластеризация ключевых слов |

### Пример: AI-чат

```javascript
const response = await fetch('/api/ai/chat', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    messages: [{ role: 'user', content: 'Сколько стоит монтаж вентиляции?' }],
    stream: true  // или false для обычного ответа
  })
});
```

### Пример: Мета-теги для городской страницы

```javascript
const response = await fetch('/api/seo/meta', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    type: 'city',
    data: { cityName: 'Химки' }
  })
});
// { title: "Монтаж инженерных систем в Химках — Freonn", description: "...", keywords: "..." }
```

## Гарантии безопасности

Каждый компонент защищён от сбоев:

| Ситуация | Поведение |
|---|---|
| `GROQ_API_KEY` не задан | Сайт работает штатно, AI-функции отключены |
| Groq API недоступен (503) | Показывается статический ответ с телефоном |
| Groq API вернул ошибку | Логируется в консоль, пользователь видит fallback |
| Запрос завис (таймаут 15 сек) | Автоматическая отмена, показывается fallback |
| Скрипт не может подключиться | Выходит с понятным сообщением, файлы не трогает |
